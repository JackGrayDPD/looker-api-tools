# looker-api

## addGroupToGroup
Adds `child_group` as a member of `parent_group`. Run `node addGroupToGroup --parent_group=20 --child_group=21`. Run `node addGroupToGroup --help` to see more options.

## updateUserAttributeGroupValue
Run `node updateUserAttributeGroupValue --group_id=GROUP_ID --user_attribute_value=USER_ATTRIBUTE_VALUE --value=NEW_VALUE`. Run `node updateUserAttributeGroupValue --help` to see more options.
